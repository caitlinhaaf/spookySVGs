Images on the Web

Bitmap Vector
  • Made up of pixels
  • Each pixel appears as a tiny square
  • Each pixel is assigned a colour code
  • Can be edited in Adobe Photoshop, Paint
  • Uses math to draw lines and shapes
  • Can zoom in and enlarge without getting pixelated
  • Can be edited in Adobe Illustrator, or a code editor!

SVG's

  • Stands for "Scalable Vector Graphics"
  • Allows us to add or build our own graphics to pages using code
<html> <body> <h1>My first SVG</h1> <svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" /> </svg> </body> </html>

CSS

  • Stands for Cascading Style Sheets
  • Used to add styles to your website (i.e. fonts, colors, layout)
  • Can also be used to add animations
  • Syntax uses a selector (the element you want to style), and sets of properties and values.
p{ color: red; back-ground-color: orange; }

CSS Transforms

Used to scale, move, rotate, or even use 3D effects

#square{ transform: rotate(60deg); } #circle{ transform: scale(2); } #triangle{ transform: translate(100px, 200px); }

CSS Animations

Before adding an animation to an element, we have to define the animation.

@keyframes float { 0% { transform: translateY(0%); opacity: 1; } 50% { transform: translateY(-5%); opacity: .5; } 100% { transform: translateY(0%); opacity: 1; } }

CSS Animations

When applying an animation to an element, we have the define:

  • The name of the animation we want to use
  • The length of the animation we want
  • How many times we want to play the animation

#ghost { animation: float 4s infinite; }
